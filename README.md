#🚀 CodeLens AI

Decode Code. Reveal Intent.

📌 Overview

CodeLens AI is an intelligent system that analyzes messy, undocumented, or complex code and converts it into meaningful insights. It helps developers understand the original intent behind code and identify logical gaps or bugs.

🎯 Problem Statement

Understanding legacy or poorly documented code is difficult and time-consuming. Developers often struggle to:

Identify the original intent of the code
Detect logical errors or hidden bugs
Quickly understand complex implementations
💡 Solution

CodeLens AI provides an automated way to:

🧠 Extract intent from code
⚠️ Detect bugs and logical issues
📊 Score how well code meets its intended logic
💡 Suggest improvements
🔁 Provide alternative implementations
🗣️ Explain code in simple language
✨ Features
Upload or paste code in multiple languages
Intent extraction and bug detection
Smart scoring system
Suggestions and alternatives
Natural language explanation
User authentication (Login/Register)
History tracking
🧠 System Workflow
User logs in
Uploads or pastes code
Code is analyzed
System generates:
Intent
Bugs
Suggestions
Score
Results displayed in UI
🛠️ Tech Stack
Frontend: Streamlit
Backend: FastAPI
