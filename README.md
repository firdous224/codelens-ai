# 🚀 CodeLens AI  
### Decode Code. Reveal Intent.

---

## 📌 Overview  
CodeLens AI is an AI-powered system that analyzes messy or undocumented code and converts it into meaningful insights.  
It helps developers understand code intent, detect bugs, and improve logic.

---

## 🎯 Problem  
Understanding complex or legacy code is difficult because:

- ❌ No documentation  
- ❌ Hidden bugs  
- ❌ Hard-to-understand logic  

---

## 💡 Solution  
CodeLens AI provides:

- 🧠 Intent Extraction  
- ⚠️ Bug Detection  
- 📊 Code Quality Score  
- 💡 Suggestions  
- 🔁 Alternative Implementations  
- 🗣️ Simple Explanation  

---

## ✨ Features  

- Upload or paste code  
- Analyze multiple languages  
- Detect logical gaps  
- Interactive UI  
- Login & user system  
- History tracking
